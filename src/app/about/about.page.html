About About QuizCards is an easy way to start learning Chinese or study and practice on your own.

How To Use QuizCards: Each QuizCard Set includes Study Cards and Quiz Cards. First study and learn the Study Cards until you're comfortable with them. Then try the Quiz Cards to see what you've learned.

Repeat until you've mastered the Quiz Cards, then move on to the next set of cards! Currently using QuizCards version 1.0.0 Check for updates
